getXYSpectrum

src/util/getXYSpectrum.js

Retrieve the spectrum with only X/Y data that match all the selectors If more than one variable match the selector the 'x' or 'y' variable will be taken

getXYSpectrum(spectra: Array?, selector: object): Spectrum
Parameters
spectra (Array? = []) Array of spectra
selector (object = {})
Name Description
selector.units string? Units separated by "vs", e.g., "g vs °C"
selector.xUnits string?
selector.yUnits string?
selector.labels string? Labels separated by "vs", e.g., "relative pressure vs excess adsorption"
selector.xLabel string? will be converted to case insensitive regexp
selector.yLabel string? will be converted to case insensitive regexp
selector.dataType string? will be converted to case insensitive regexp
selector.title string? will be converted to case insensitive regexp
selector.meta string? object of key/value to filter meta information. Each value will be converted to case insensitive regexp
Returns
